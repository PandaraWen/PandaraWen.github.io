都是假的

发表于 2017-07-16   |   分类于 口水沫儿   |   访客 0
Yo~

儿时以为是人就能分个好与坏，
成年后才知道世界不只黑与白。

灰色才是常态，
眼前尽是雾霾。

政治家跟你说世界太平你一定要乖乖，
稍有不慎触碰火线你就跟世界说拜拜。

爱豆说声快跟我一起来来来，
粉丝举臂傀儡般齐声嗨嗨嗨。

世界观的塑造是一直迭代，
没有必要过早向世人表态。

by MC烧饼


我们所生活的世界其实并不那么真实。处处都是雾霾，或者是假象。造成这些假象的，可能是政治家愚昧人群的手段，也可能是我们自身的限制。这些假象，影响着我们对世界的认识，正如我认为，我们都生活在一个充满虚幻假象的世界里。


0x01 自身的限制


我们对这个世界的认识，是一系列观念的集合，世界观的塑造过程，就是观念之间的碰撞、新生、迭代、消失。而炼成这些观念的容器，就是我们成长的环境。每个人的成长环境都不相同，塑造了我们各自不同的世界观。虽然我们都没有显式表达过，但其实人人心里都会具有这么一个潜在观念：世界就是我认为的那样。


所以当某些超出我们认知的事物出现时，那些在我们的成长环境中从来没有出现过的黑天鹅突然显露的时候，我们会觉得自己的世界观被颠覆了。我们对世界的认识又更进了一步，或者将之前的世界观定义为假的，过时的，将它推翻。但我们不知道，当下构建起的这个世界观，会在什么时候又被推翻。


0x02 外部的假象


当某篇表达作者自身世界观的网络红文在朋友圈被刷爆，十几万的阅读者中，不知道有多少比例的读者的观点被文章「调整」，在无声无息之中变成作者的盈利工具；电视台无终结滚动重复播放的保健品广告里，不知道又有多少中老年人为这些玄学产品贡献了资源；某个 idol 振臂一呼，一大批粉丝就像傀儡般开始整齐划一的行动；诡辩者给你灌输几个无法证伪的观念，再用自己的一套自圆其说的观念给你解释他所看到的世界，让你仿佛看到新世界的大门仿佛被打开；身居高堂的大臣用手段玩弄蝼蚁般的群众，看着蝼蚁们在自己的掌控下蠕动……


0x03 End


今天所认定的，可能明天就被揭露是假的；你所认定的某种生活方式，某天可能彻底被久未谋面的朋友彻底敲碎。带着这些类似的观念生活，不得不说会让人踌躇不前，为之四顾，为之茫然。万一，我说的这些都是假的呢。
