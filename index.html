是否也活得用力过猛

发表于 2016-08-20   |   分类于 口水沫儿   |  

是否也曾经在网上看到某一篇减肥/健身的帖子，然后心中燃起熊熊火焰，暗暗发誓”我也一定要活出自己想要的人生”，”我明天就去健身/跑步/节食，做不到我就是孙子！”。然后当时刻来临时，有些人发现自己仍然跟昨天没有什么区别，于是就会给自己下了一个定义：我果然还是没救了啊，我果然做不了这件事呢。而有一部分人会付出行动，比如办了一张健身卡，每天哼哧哼哧地举铁。但是一段时间过后，发觉自己并没有变成想象中的六块腹肌，于是焦虑情绪渐渐涌上来，心里冒出来一把声音：”反正还是做不成了吧”。这把声音慢慢加固，让行动渐渐松懈，这又反过来增强了那把声音，恶性反馈，最终让宿主走向了放弃。这样子是不是活得有点用力过猛了。

Multipeer Connectivity

发表于 2016-08-11   |   分类于 你丫才码农   |  

随着互联网技术以及各种云服务的发展，设备之间的连通变得日益重要。


在这个移动计算的时代，无论在工作还是娱乐，多人合作的可能性变得空前巨大；也是这个注重隐私却充斥着监控的时代，对安全通信的需求也变得空前巨大；同时也是这个充满连接的时代，即便是随处可见的生活小物件也拥有者大方异彩的前景。


所以，iOS7 引入的 Multipeer Connectivity API 可能会成为最优秀的连接设备方式。它让开发者可以重新考虑移动 APP 的构建，重新定义了连接的可能性。
