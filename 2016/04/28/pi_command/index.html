修改用户名

Linux 更改用户名不像Windows 重命名那么简单，首先你要知道在Linux中创建了一个用户，除了生成UID外还会有一个GID，当你对原用户名修改后，别忘了对GID也进行更新。

修改用户名：
以root身份登录，然后使用下列命令进行修改

-l 修改用户名
-d 修改登录后使用的路径
-m 修改登录名称的同时将目录名称一同修改
