《死神的精确度》

发表于 2016-02-08   |   分类于 说书人   |     |   访客 0

在人类的角度，这个死神非常有趣。而在他的角度，人类认为他有趣反而使他感到郁闷。他(or 她？死神应该没有性别吧)对人世间事情漠不关心，唯独享受人类的音乐。与“工作对象”之间看似无厘头的无心对话，又带出许多哲思。这本书以这位死神为主线，描写了6个人的故事。这本书给我的感觉，跟《解忧杂货铺》很像，都是在几个短篇故事中融入了人情冷暖，引发哲思。当然，还是杂货铺更加“震撼”



1.死神的精确度
这位死神懂得人类的语言，但不是十分精通，于是有时无意间会创造出一些高级撩妹技能：

她夹了一口鸡蛋料理吃下去后，嘟囔了一句：“我长得太难看了”。“难看？”我一时真的没领会过来，于是眯起眼睛，拉远了距离看着她说，“不会，很容易看到，不是很难看啊。”


这是本书6位人物中，死神唯一放行的姑娘。或许是她声音好听，且被音乐制作人看中，有成为歌星的潜力吧。


再摘录一段翻译得不错的文字：

回过神，可能是雨越下越大的缘故，发现雨水落在地面上，如大珠小珠落玉盘般急切，仿佛正在催促我快点下结论。


2.死神与藤田
藤田所坚守的黑道，等同于游离于法律之外的正义，犹如高谭市的蝙蝠侠。但这无疑是让他成为鹤立鸡群的特异点，就像一般情况下，特立独行的人事物总会收到排斥，难以被融合。


这篇故事后面的小转折挺有趣，虽然藤田最终的报告是“可”，但他的对头还是比他先死一天，因为千叶发现了负责调查藤田的对头的死神，比千叶早开始调查一天。


摘录一点描写千叶心理的有趣文字：

“我不有趣。”又被人说有趣！我明明是在拼命地认真工作，却被说成“很有趣”，这让我情何以堪。


3.暴风雪中的死神
这个短篇非常像《东方快车谋杀案》——涉及的嫌疑人全都是凶手，只为杀死一个共同的仇人。本来计划应该能够顺利进行，但是因为千叶为了调查工作的出现，让计划节外生枝。可以当做一篇短短的悬疑小说来看。


4.死神与恋爱
男主角相当有趣，故意戴着一个脏脏的眼镜，来掩盖自己俊俏的容貌，企图排除掉那些因为他的外貌而接近他的人。摘录一段：

“我是那种想到什么就说什么的人。人生不知道什么时候就结束了，该说的时候就要说出来，管它是不是唐突，你说呢”


5.旅途中的死神
这篇应该是我最喜欢的故事，转折挺合理，而且早埋下伏笔，互相呼应。男主角一开始企图杀死的，认定他是绑匪一份子的瘸腿老汉，到后来被发现其实是自己的真正救命恩人。而男主角此时已经因为这件事从童年长成了青年，而且形成了畸形的心理，刺伤了自己的母亲以及刺死了一名不相识的路人。摘录一句：

“人活着的大部分时间都算不上在生活，不过是虚度光阴而已。”


6.死神与老妇人
这个短篇中的女主角也是故事4中的女主角。围绕她身边的不幸事情太多了，不幸到自己的儿子都嫌弃她。最后有机会可以见孙子，但是为了不认出哪一个是孙子，怕其遭到不幸，老妇人让千叶找了几个年龄相仿的年轻人到店里，这样便能跟她的孙子混淆。


就算这位死神不喜欢，我还是要说，他很有趣。他总是一副波澜不惊的样子，内心又总是看透一切，超越生死(这是自然)。心里说的那些调侃话语也非常可爱。每一篇故事，都可以带给读者一些思考。
