纯真的脑洞

发表于 2015-07-20   |   分类于 小废画儿   |     |   访客 0

上一年末回到本来的公司工作。创业公司人少，一个人顶两个人用，除了iOS开发，我还兼设计。因为本来对设计感兴趣，所以毛遂自荐做起了项目里的美工（并不敢自称设计师）。但是牵扯到工作，兴趣也不那么有滋味了。这个日后再娓娓道来吧。
设计做多了，脑洞也慢慢地越开越大了。有一些已经丢了，下面是一些能找到的脑洞：






1.机器人
某周日公司加班，在老地方抽烟时看见这个AI独自迎着血红残阳
当年人类与机器人之间的激斗，在它空洞的眼睛里没了踪影
习惯寂寞
想必十年对它来说早已只是一瞬
