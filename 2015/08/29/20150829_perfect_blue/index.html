距离精神分裂仅一步之遥

发表于 2015-08-29   |   分类于 电影铺   |     |   访客 0

脱离现实的精神嫁接，轻则导致厌恶现实，重则会导致人格分裂

不久之前做了个决定：每看完一本书，一部漫画，一部电影，都尽可能留下自己的感想。无论文字多少，无论感想是高深还是肤浅。为的是给将来的自己看，在某个时间点回首能够看到来时脚印。

看今敏的第一部电影是《红辣椒》(又名Paprika，2006年)，当时就被电影里面节奏畅快的音乐以及色彩绚烂的浓浓日本风情画面吸引住。后来隔了一段时间，再看今敏的另外一部高评价动画电影(其实今敏的动画电影作品评价都挺高)《未麻的部屋》（又名Perfect Blue，1997年），对他的佩服又深了一些。然而我今天才知道原来他几年前就已经去世了，如果他仍在在生，想必会有更多佳作。

如果硬要贴标签的话，这算是一部悬疑片。
剧情中有很多看上去像是梦境与现实之间的交叉衔接，让我一时之间搞不清楚到底是哪部分是梦境，哪部分是现实。虚虚幻幻真真假假，让我烧死了不少脑细胞，这点跟《红辣椒》倒是很像。

电影里让我印象深刻的地方有两个：

未麻对过去的留恋，引来了自己幻想中的虚拟形象

未麻因为想要在事业上更上一层楼，脱离偶像圈，进入演员圈子。但是很多事情总是表面风光暗藏血泪，演员也如此。表面看上去粉丝前呼后拥，名扬万里，风光无限，但实际上不知道要走过多少曲折，付出多少努力，忍受多少困难，拼得头破血流去争取在电视多那么一点的戏份。其中还包含不少人事杂乱，勾心斗角。
未麻一开始也遇到这些林林总总的困难，特别是要适应从台上受粉丝簇拥的偶像掉落为被人气演员光芒所淹没，在一旁板凳冷坐的演艺新人，这天与地之间的差距。为了增加自己的露脸机会；同时也因为自己作为一个新人，好不容易争取来剧本的修改，就算是强奸戏也硬着头皮接下来了。不得不说这一幕看得可真算是揪心，声优喊得真够撕心裂肺的。这也成为了故事的开端。
未麻开始后悔，开始怀念从前作为偶像的华丽丽的日子。此时镜子里面就出现她幻想出来的，曾经作为偶像而存在的那个虚拟的自己。两个未麻在争论，或者说是争斗。这其实可能是未麻内心在挣扎，挣扎着自己的选择到底是否正确，以及对现状的不安与怀疑。

留美将自己的愿景嫁接在别人身上，终于导致人格分裂

留美一直反对未麻的转型，在看见未麻演出强奸戏的时候更是伤心至落泪，起身离开片场。留美一直希望未麻保留偶像的身份，大概也跟她自己过往的经历有关（从墨镜大叔那句：这已经不是留美你活跃的那个时代了）。无法接受偶像无法像过往一样可以很好地生存的当下，在结合未麻放弃偶像转型演员，大概让留美心里那个曾经熟悉的世界出现了裂缝。而未麻的强奸戏则应该是将它完全粉碎的最后一根稻草了。
为了保留住那个自己所坚守的已经逝去的现实，留美拼命伸手抓取，最终她找到的那个保留自己幻想世界的方法就是，让自己成为未麻，那个过去作为偶像而存在的未麻，现在的这个“变了质”的未麻，就让她去死吧。
于是，她亲手杀死了一个又一个让未麻逐步在演员路上走远的人：编剧、写真（果照）摄影师、墨镜大叔，并且将凶器放到未麻房间里，企图制造她的精神混乱。
最后到了电影的高潮，在留美那个装饰得跟未麻的一模一样的房间里，留美穿着未麻的演出服，从厨房里走出来，准备将自己所不接纳的现实——未麻——从这个世界上排除。不得不说，画面太美不敢看


赶紧看几张未麻的写真(果照)压压惊

最后当然失败了，这场由留美自己内心对现实强烈的抗拒排斥，所最终引发闹剧，终于落幕。

这个画面争议不少，有人认为未麻对着镜子说这句话，是因为其实这是个谎言，其实未麻才是所有事件的凶手。这个嘛，今敏自己都说没有正确答案，所以，我们每个人都可以有自己最满意的解释。但是，不要拒绝别人不一样的解释，否则变成留美那样就不好了。

题外话，红辣椒跟这部影片都不排斥使用色情、血腥等元素来拉取眼球，不知道这是否今敏的作品的共性呢。下一部让我看看《千年女优》。顺便一说，女优指的是女演员，并不是特指AV女优，太多人混淆了。
