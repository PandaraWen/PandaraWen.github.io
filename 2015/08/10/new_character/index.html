周末就该安静地画画

发表于 2015-08-10   |   分类于 小废画儿   |     |   访客 0

这个周末我把自己重构了一下

这次是6个半小时的静坐



历史总是如此的相似。
与上次鼓捣blog一样，某天早上习惯性的看了一下自己的blog之后，心里蹦出一个想法：嗯，缺一个“about me”页面。可是应该从哪里着手呢。恰好最近在看AE的一些教程，受到里面的一些启发，我想我应该先给自己重新鼓捣一个二次元版本。于是我提起了在文具店特意挑的“最重的自动铅笔”，抽出我的牛皮纸方格本，一手翻开空白页。微微地直了直腰，稍稍地挺了挺胸，轻轻地抬了抬头收了收下巴，为之四顾，为之踌躇满志。一会儿之后，本子上的手稿是这样的：


我看着这个东西。挖槽！这是什么鬼！吓死本BB了！这圆圆的像被截肢了一样的手脚是什么鬼！？！？还有眼睛闭着就像瞎子，这简直就是无可挽救的重度伤残人士啊我的老天！还有那个一点都不Q的Q版人物比例，失败中的失败。可是它就是出自我手，我必须承认这个铁打的事实。我想我有必要练一练手绘Orz。
这个稿子可取的地方就只有发型。于是我把发型保留了下来。之后习惯性的上Dribble寻寻觅觅，遇到了某位大神的作品，心生欢喜，可以作为参考：

我打开躺椅钻了进去窝在里面，随即进入flow模式。时间流逝得贼快几乎感知不到，在6个半小时之后，从下午2点到晚上8点半，我鼓捣出一个大智若愚不羁放纵爱自由的Avatar：


看呀，他那眼睛：


表面上看起来好像空无一物。可是，要知道，


当所有七彩缤纷的可见光汇聚为一起的时候，白光从中而生。

大智若愚是也。
我欣慰地笑了，这应该可以用很长一段时间而不被我厌弃。接下来就是在下个周末，再花上6个小时，给他加上看似颓废实际从容儒雅的行走动画，鼓捣自己的about页面。貌似风格跟blog的基调好像有点不同，不过：


步履不停，探索不止

试错趁早，搞搞看吧。
