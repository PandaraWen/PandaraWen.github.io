煎饼侠

发表于 2015-09-26   |   分类于 电影铺   |     |   访客 0

如果你喜欢听故事，那么这是一部好电影。


之前看过大鹏写的一本励志书，《在难搞的日子笑出声来》。且不论里面描述的事实有多少成没有经过修饰的——或者说里面透漏的是事实的几成——这本书描写的的确是一个很励志的故事。大多数我看书的时候只求真实，对于虚构类的非写实类的一概反感，特别是当我感觉到有很浓重的矫情成分时就是弃读（关于这是否为一种心理上的缺陷：缺什么嫉妒什么因而反感什么，就是后话了）。但是大鹏的这本书我算是一口气读完了。而这次看《煎饼侠》，感觉就像再读《难搞》一般，一个酣畅淋漓的故事，一个可能落入了某种俗套但是确实能让人产生共鸣，能够承载人们美好想象的故事。


煎饼侠跟《屌丝男士》的路子很像，通过大牌明星的客串以及剧情反转来迎合观众的口味，能给人带来新奇感刺激感。而且同样地，所有明星在电影中都是以自己的身份出现，让这个电影故事又多了几分现实性。在电影里看到大牌明星本色出演并且被恶搞，无疑是一件很爽的事。


故事本身没啥好挑剔的，主角从巅峰因为某些负面情绪导致事故，继而衰落，接着奋斗，认识一帮人，收获一段情。最后来个大团圆结局。有血有泪，有笑有苦。不得不说，古惑仔出场的画面很有感染力，估计会让不少80 90汗毛树立，鸡皮疙瘩都起来了。


从故事内容上来评分，我给8.5分


本来想再写点什么，不过因为这是一个故事，还是不写了。
