你个鸟人

发表于 2015-09-03   |   分类于 电影铺   |     |   访客 0

瑞根还有他那把碎碎念的声音，好像我。


我想应该每个人的内心都会想瑞根一样，有一把碎碎念的小声音，时不时跑出来在脑子里奏响熟悉的旋律。而这把声音，往往还可能就是最真实的想法，最本能的冲动，最原初的欲望，最人性的恶魔。

曾经我极端抗拒这把声音，排斥，憎恨，讨厌。每当那把声音冒出来，我可能会陷入一小阵子的迷失。当我回过神来，这时候碎碎念的声音变成了圣人模式：“我怎么会有这个念头？”“我怎么会冒出这个想法？”“这可真是丑陋的idea啊！”。突然想起来，这就是小时候看肥皂剧经常看到的“天使与魔鬼”。只不过恶魔的声音往往是第一时间出现，而天使的声音紧随其后罢了。

在这两把声音的夹缝中求生存不容易。有时候会坦然的看看自己心里最肮脏的角落，然后了解接纳但不排斥，就让它在哪儿。不过更多的时候是在那把黑色阴沉的声音制造出来的幻想场景中迷失。之后当我回过头来，白色圣洁的声音随后而到。

就像电影中瑞根一样，有时候黑色的声音占了上风，造就出来的幻觉甚至有可能影响我现实的行为，以至于当我猛一回神发现自己在自己的幻想中走了不少距离，滑稽得让我笑出声来。

瑞根还算幸运，打开心门接纳心里的声音，拥抱它，收获自己的inner peace，最后还得到了事业收成。而在这之前，他正在为自己所做的一切而迷茫，他正在试图通过这个话剧试图证明自己，试图将自己从以往的birdman光环抓出来，重新用另外一种成功来证明自己。

或者这部电影试图讽刺好莱坞最近的热门大片以及大众的审美趣味。它讽刺当下很多电影仅仅依靠特技噱头炫酷暴力血腥的刺激元素来吸引观众获取成功。包括他的底裤裸奔还有搏击俱乐部男主的话剧勃起，为他们各自带来了相当高的关注度；以及瑞根的女儿将瑞根的住院照po上推特，都应该是试图对当下大众的“低级趣味（？可能）”进行讽刺。尾瑞根接纳了碎碎念的建议，其实也就是自己内心的真实想法——同样使用噱头血腥来获取关注度，最后居然大获成功，而这是否为一种反讽？
