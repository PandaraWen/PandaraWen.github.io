Clean my mac

发表于 2015-09-21   |   分类于 口水沫儿   |     |   访客 0

不能掌控自己macbook的硬盘的人谈何扩展生命宽度。说的就是我


眼看着macbook剩余空间越来越少，装个xcode7都要先把原来的删掉才能腾出空间，我特么的悲催啊。想换个容量大点的ssd，京东一下:




无言独上西楼，月如钩。 寂寞梧桐深院锁清秋。
剪不断，理还乱，是离愁。别是一般滋味在心头。


无奈只好自己动手清清垃圾，进入根目录，查了一下自己的用户文件夹，一共60多G，占去了硬盘的一半。好吧，开始清理，下面记录一些容量比较大的并且可以删除文件跟文件夹


/Users/Pandara/Library/Application Support/MobileSync

这是使用itunes(?)备份手机之后的备份文件所在之处，占去了我7G


/Users/Pandara/Library/Developer

这是跟xcode相关的文件夹，占去了我37G。。。
里面有两个文件夹：


CoreSimulator
不知道是什么，看名字好像跟模拟器有点关系。占去了我1.5G


Xcode
占去了我36G。。。里面有不少东西，包括archive


不管三七二十一，全删了，当我今天是头一天用Xcode就好了。。
另外系统文件夹的东西我就不敢动了。结束。

——————————后记——————————
记录一下这样做的副作用：

  1. hexo提交代码的时候提示我需要先同意xcode的许可协议

  2. 用cocoaPods导入的库不能用了，要重新安装cocoaPods(我顺便更新了一下)

  3. 然后安装(更新)依赖
