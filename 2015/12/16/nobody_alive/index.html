《无人生还》

发表于 2015-12-16   |   分类于 说书人   |     |   访客 0

十个人
一座岛
无人生还

前不久读完《东方快车谋杀案》，这是我读的第一本阿加莎的作品。而如今读这本《无人生还》，阅读快感比东方快车谋杀案强太多了。


整本书95%的内容都是无法解释，不可理喻的离奇谋杀案件。读书时，看着里面的角色一个接着一个死去，心中集聚的悬疑感越来越强，勾起我巨大的阅读渴望，真算得上是手不释卷。不得不说，看着书中的角色逐步死在离奇的谋杀手法之下，还是挺有快感的。


读这本书时，我脑子里的画面全都是柯南里面的换面。。连人物风格都是柯南的绘画风格。。大概是这种“孤岛谋杀案”跟柯南里面的不少剧情太过相似了吧，又或者说柯南借鉴了这部作品不少内容。值得一提，书中的关于十个小兵的童谣添加了不少恐怖感，营造气氛功不可没。


除了人名难以记忆这一点，我对这部作品真的没有什么好挑剔的，基本上是顺畅如丝般地忘我阅读。在最后解开谜底的案情大抽丝上，解释得也很在理，密不透风，也难怪它能在豆瓣上得到 8.9 跟 9.0 的高分了。这部小说可以说是开创了一个流派了。


或者可以从这本小说里面窥探到一些人生鸡汤：若要人不知，除非己莫为。
