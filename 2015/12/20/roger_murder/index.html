《罗杰疑案》— 抽你一大嘴巴子

发表于 2015-12-20   |   分类于 说书人   |     |   访客 0

剧透哦，千万不要点进来哦。


看这本书最大的感受，就是，脸被猛地抽了一下。作者跟读者卖了最大的一个关子：整本书以谢泼德医生的视角书写而成，并且隐去了一些细节。当然这些细节并没有构成最后破案的最大关键线索。当所有线索最终都指向最终的凶手，菠萝侦探终于在最后将凶手指了出来，真相只有一个，凶手就是你——谢泼德医生。当时我的心情是：


阿加莎跟我开了一个天大的玩笑。


回顾书中人物性格，各有特点，各有缺陷：

  • 外表端庄而实际上为了她的光鲜而欠下一屁股债的阔太；
  • 天真可爱的少女，在被逼到无可奈何时，也会做些偷鸡摸狗的事情；
  • 一个警官，也会经常为了自己的面子耍些小心机，意图在某些场合挽回一下自己的颜面；
  • 勇猛而无所畏惧的军官，在自己的心爱的人面前也会局促不安脸泛红晕；
  • 最后最大的人格缺陷的角色，毫无疑问是本书的主角，谢泼德医生。


对于谢泼德医生，可以用书中的这段话来形容：

比方说，有这么一个普通人，一个非常普通、心中全无杀意的人。他的骨子里潜藏着某种性格缺陷——藏得很深很深，迄今为止都没有人发现，或许一辈子也不会表现出来——那么他将体面地走完人生之路，受到所有人的尊敬。但假设他因为某些缘故而陷入困境——也许不至于如此，也许他是偶然窥见某个秘密——对某人而言性命攸关的秘密。他的第一反应式说出来——旅行诚实的公民义务。但是他潜在的性格缺陷可能会开始冒头：这可是个发财的好机会——天降横财啊。他想要钱，渴望搞到这笔钱，而这笔钱又唾手可得。他不必付出代价，只需要保持沉默，不揭发那个秘密。但这只是开始。他对金钱的欲望与日俱增，渴望弄到更多的钱——越多越好！眼前这座已开采的金矿令他陶醉，他的贪念不断膨胀，贪婪扭曲了他的人性。如果对方是个男人，那尽可以随便压榨——但对于女人，逼人太甚是大忌。因为女人有一种说真话的强烈本能。因为女人有一种说真话的强烈本能。
有多少丈夫蒙骗了妻子一辈子，把秘密带进坟墓。而又有多少不忠的妻子对同样不忠的丈夫坦白，从而毁了自己的一声(这比例是女比男多)。一旦被逼得走投无路，她们就会不顾一切后果(当然，时候免不了又会后悔)，忘掉个人安危，只图一时痛快，就倾吐全部真相。


有些时候，人非尽善，也非尽恶。每个人心中都有善与恶两种思想，它们会互相博弈互相试图抢占大脑的控制权。某些时候，可能恶的思想会占据了大脑，而下次，处于愧疚或者某些原因，善会重新占据大脑做决断。一些大奸大恶杀人成性的人，当然，可能恶会占了大多数。其实说这么一段话我很没有底气，因为善与恶这两个概念本身就非常局限，况且它们非常具有局限性，时间局限性以及空间局限性。真要追根究底，所有的善都是从自身利益出发，就像所有的恶也是从自身利益出发。不过一个是多赢，另一个是独利。
